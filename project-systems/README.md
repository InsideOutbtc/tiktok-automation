### 🚀 Quick Start

1. **For New Projects:**
```bash
./init-new-project.sh my-project
```

2. **Load Framework:**
Open `constitutional-ai/AI_AGENT_PROTOCOL.md` in Claude

3. **Generate PRP:**
```bash
cd prp-framework/commands
node prp-generate.js base "initial-setup"
```
